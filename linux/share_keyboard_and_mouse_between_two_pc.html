两台电脑共享一套鼠标和键盘

2018-11-28

工作时，我有两台电脑，两个显示器。有时用两台电脑时，经常需要两套键盘和鼠标进行切换。 用久后难免觉得麻烦。

我两台电脑在同一个局域网中，google了一下，找到 x2x 这个软件。

 +-----------------------+                                           +-------------------+
 |                       |                                           |                   |
 |         client        | ssh -XC user@serverip  x2x -east -to :0.0 |     server        |
 |                       |------------------------------------------>|    install x2x    |
 |                       |                                           |                   |
 +-----------------------+                                           +-------------------+

使用很方便，在server端装好后，在client端执行一下下面这条命令就可以了。

ssh -XC user@laptop  x2x -east -to :0.0

这样我的server就的屏幕就在client的east，也就是右边了。 鼠标移动到右边应该就可以操作server这台电脑了。再移动回来就是client电脑。
